Room Adventure - a java project for CSC 132

This is my individual Room Adventure project completed for class.  
The game includes four connected rooms, items, grabbables, and three custom features.

Features:
DROP: drops items from inventory into the current room  
Inspect: View detailed info about items you're carrying  
USe: Special items with room interactions (e.g., powering a robot with a battery)

Rooms:
Lobby
Robotics Lab
Server Room
Storage Closet
